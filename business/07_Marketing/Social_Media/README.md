# Social Media Workspace

This folder is the planning layer for Instagram and Facebook content.

## Purpose

- Keep strategy, calendars, and caption templates in the repo
- Keep actual image/video files on your desktop, not in the CRM and not in git
- Use the CRM `Social` page to track what is drafted, ready, and posted

## Recommended local asset folder

Create this on your desktop outside the repo:

`/Users/rajsingh/Desktop/Fitterverse_Social_Assets/`

Recommended structure:

- `2026-06/`
- `2026-07/`
- `raw/`
- `exports/`

## Weekly workflow

1. Add ideas in [Post_Ideas_Backlog.md](./Post_Ideas_Backlog.md)
2. Move selected ideas into [Monthly_Content_Calendar.md](./Monthly_Content_Calendar.md)
3. Draft captions from [Caption_Library.md](./Caption_Library.md)
4. Save media files in the desktop asset folder
5. Log the final local file path in [Asset_Register.csv](./Asset_Register.csv)
6. Create or update the post in the CRM `Social` planner
7. Publish manually in Meta Business Suite or the native apps
8. Mark the post as `posted` in CRM

## Rules

- Do not store videos or image exports in this repo
- Store only text, planning templates, and asset references here
- Keep file names date-based so the asset register and CRM stay easy to match
