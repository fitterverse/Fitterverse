# Posting Checklist

## Before publishing

- Caption reviewed
- CTA included
- Hashtags checked
- Local asset path confirmed
- Cover frame checked for reels
- First line strong enough for hook
- Links or offer details verified
- Brand spelling checked: `Fitterverse`

## At publishing time

- Publish to correct platform(s)
- Add first comment if needed
- Pin comment if planned
- Confirm thumbnail/cover looks right
- Check final post is visible publicly

## After publishing

- Mark post as `posted` in CRM
- Add final live URL in notes if useful
- Track early comments and DMs
- Reuse winning hooks in future posts
