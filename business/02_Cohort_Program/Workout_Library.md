# Workout Library — Outline

## Production checklist

For v1, we need to produce or license:

- 9 unique workouts × 2 versions (home, gym) × 3 levels (beginner, intermediate, modified) = ~54 short videos
- Each video: 60-90 seconds demo + a written breakdown
- Filmed phone-vertical (9:16) so they double as Instagram content

## Format for each workout entry

```
WORKOUT: <name>
LEVEL: beginner / intermediate
DURATION: <minutes>
EQUIPMENT: <list>
WARMUP: 3 min (specific moves)
MAIN BLOCK: <exercises with sets x reps x rest>
COOLDOWN: 2 min (specific moves)
MODIFICATIONS:
  - knee-friendly: <what to swap>
  - back-friendly: <what to swap>
  - 10-min express: <what to drop>
COACHING CUES: <3-5 form cues>
```
