# 90-Day Roadmap

## Goal
By Day 90, run 3 successful cohorts, generate ₹2.5L+ revenue, and have 25+ documented transformations. Use that to decide v2 direction (subscription vs corporate vs scale cohorts).

---

## Days 1–15 — Foundation

**Output:** Landing page live, payment working, cohort 1 ready to fill.

- [ ] Buy/configure WhatsApp Business API via AiSensy or Interakt
- [ ] Set up Razorpay account and payment link
- [ ] Build landing page (Next.js + Tailwind, see `08_Product_Tech/MVP_Build_Plan.md`)
- [ ] Build onboarding assessment form (Typeform v0, custom form v1)
- [ ] Write 21-day curriculum (see `02_Cohort_Program/`)
- [ ] Write all WhatsApp daily messages for 21 days (see `05_WhatsApp_Ops/`)
- [ ] Hire 1 part-time coach (nutritionist + fitness background)
- [ ] Set up admin dashboard (Notion + Airtable v0, Retool v1)
- [ ] Record 30 short-form reels for Instagram (see `07_Marketing/Reels_30Day_Calendar.md`)
- [ ] Define champion program rules (see `03_Champion_Program/`)
- [ ] Write disclaimers, T&C, refund policy (see `10_Legal/`)

## Days 16–30 — Cohort 1 (validate)

**Output:** 30 paid users, 60% completion, 5 transformation case studies, NPS > 40.

- [ ] Open enrolment for Cohort 1 (target 30 seats)
- [ ] Run paid ads ₹15K budget — Meta lead ads + retargeting
- [ ] Founder personally onboards every user with a WhatsApp call
- [ ] Daily check-ins via WhatsApp (manual or semi-automated)
- [ ] Mid-cohort review at Day 11
- [ ] End-cohort: collect before/after, weight, photos, testimonials, NPS
- [ ] Announce top 3 winners (cash + free next cohort)
- [ ] Founder personally interviews 10 users for product insight

## Days 31–60 — Cohort 2 (improve + grow)

**Output:** 75 paid users, 65% completion, 12 case studies, ₹75K revenue.

- [ ] Apply lessons from cohort 1 — fix top 3 friction points
- [ ] Add referral system (referrer gets ₹200, referee gets ₹200 off)
- [ ] Launch SEO content — 10 blog posts, 4 calculators (see `07_Marketing/`)
- [ ] Open Cohort 2 enrolment (target 75 seats)
- [ ] Increase ad budget to ₹40K, add Google Search ads
- [ ] Hire 1 more coach (1 coach per ~40 users)
- [ ] Build proper coach dashboard (Retool or custom Next.js admin)
- [ ] Start documenting Champion winners as 90-second video case studies

## Days 61–90 — Cohort 3 (scale + decide)

**Output:** 150 paid users, 70% completion, 25 case studies, ₹1.5L revenue, decision on v2.

- [ ] Open Cohort 3 enrolment (target 150 seats, possibly split into 2 batches)
- [ ] Launch first creator collaboration (1–2 mid-tier fitness/nutrition creators)
- [ ] Pilot a B2B corporate version with 1 friendly company (50 employees, 21 days)
- [ ] Decision week (Day 85–90):
  - Path A: Subscription on top of cohort
  - Path B: Scale cohorts to 500+ users/month
  - Path C: Lean into B2B corporate as primary revenue
  - Most likely: A + C

---

## Key risks and mitigations

| Risk | Mitigation |
|---|---|
| Low completion rate kills referrals | Keep cohort small in v1, founder hands-on, hire coach early |
| WhatsApp API gets flagged | Use BSP (AiSensy/Interakt), follow opt-in rules, never spam |
| Refund requests | Clear refund policy: 50% refund up to Day 3, 0% after Day 3 if onboarded |
| Coach quality | One founder-trained coach in v1; SOP-led, not freelance dependent |
| CAC blows up | Start with founder's network + organic Instagram before paid ads |
| Medical liability | Disclaimers in onboarding, exclude pregnancy/eating disorders/uncontrolled medical conditions, partner doctor for escalations by cohort 3 |
